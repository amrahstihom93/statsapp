# Dashboard Features & Tutorial

The StatsApp Dashboard module allows you to consolidate your insights into presentation-ready reports. You can combine saved Charts and saved Analytics side-by-side into custom layouts.

## Key Features
- **Multi-Sheet Support**: Group your insights into paginated sheets, similar to an Excel workbook.
- **Custom Layouts**: Select between different grid layouts (e.g., 2x1, 2x4) to control how data is displayed.
- **Orientation Modes**: Choose between Portrait and Landscape types depending on how you plan to export or present the data.
- **Mixed Elements**: Combine visual elements (`Charts`) with tabular summary data (`Analytics`) in the same view.
- **Narrations & Subtitles**: Add custom text and subtitles to your charts to provide context to your audience.

---

## Step-by-Step Tutorial: Creating Your First Dashboard

### Prerequisites
Before creating a dashboard, ensure you have:
1. Created and saved at least one **Process**.
2. Run and saved at least one **Visualization** (Chart) or **Analytical/Statistical Summary**.

### Step 1: Initialize the Dashboard
1. Navigate to the **Dashboard** section from the main home menu.
2. In the **Name** section, enter a descriptive *Dashboard Name*.
3. From the dropdown, select the *Process Name* this dashboard belongs to.

### Step 2: Configure the Layout
1. Click on the **Layout** section to expand it.
2. Select your desired **Dashboard Layout** (e.g., `2x1` for two large elements, `2x4` for a denser grid).
3. Select your **Dashboard Type** (`Landscape` or `Portrait`).
4. Click **Submit** to lock in the layout and reveal the Sheet configuration.

### Step 3: Populate Sheet Layout Elements
Once the layout is submitted, the **Sheet Layout** configuration will appear. Here you map your saved insights to the grid.

1. **Select Element 1**: Choose the type of data you want in the first block (`Charts` or `Analytics`).
2. **Select Data 1**: A dropdown will populate with your previously saved Visualizations or Statistical summaries. Select one.
3. Repeat the process for **Element 2** and **Data 2**.
4. Click **Submit** to render the Sheet.

### Step 4: Add Multiple Sheets (Optional)
If your layout becomes too crowded, you can distribute your insights across multiple pages.
1. At the top of the rendered dashboard area, click the **Add Sheet** button.
2. A new tab (e.g., `Sheet 2`) will appear.
3. You can configure new elements specifically for this sheet.

### Step 5: Add Context (Narrations and Subtitles)
To make your dashboard presentation-ready:
- **Add Subtitle**: Use the subtitle modal to attach brief contextual headers to specific charts.
- **Add Text Narration**: Use the narration modal to add a paragraph of text explaining the findings or anomalies seen in the adjacent data block.

### Step 6: Export
Once you are satisfied with the multi-sheet layout and narrations, use the **Export** function to generate a consolidated multi-page report of your findings.
